# user-api

# Objetivo

Criar uma API de cadastro de usuarios para aperfersuar os estudos e as qualidades de desenvolvimento.

# Tecnologia

  - Spring
  - Kafka
  - Hexagonal
  - MYSQL

# Fluxo

Cadastros de usuer

  - user:
      - nome
      - sobrenome
      - CPF
      - Data
      - Edereco:
          - cep
          - rua
          - bairro
          - estado
          - cidade
          - complemento

1) Ao cadastrar um usuario tem que postar no topico do kafka
